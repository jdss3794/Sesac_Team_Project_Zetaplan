* 각 샘플 데이터 파일(.json) 사용 영역 정리

*json 파일명 정리
- newsList.json : 뉴스 리스트 데이터 파일
- tradeInformation.json : M&A 거래 정보 게시판 데이터 파일
- qa.json : Q&A 게시판 데이터 파일
- techTrade.json : 기술 거래 리스트 데이터 파일
- government.json : 정부지원사업 참여모집 게시판 데이터 파일
- policyInfo.json : 정책자금 기업지원 정보 데이터 파일
- consulting.json : 컨설팅 실적 게시판 데이터 파일

*_detail : 각 게시판 상세 페이지 데이터 파일
